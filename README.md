# Financial-Risk-Analysis-and-Predictive-Modeling-Using-Machine-Learning

This project focuses on analyzing customer financial data and building a framework for financial risk classification using machine learning techniques. The dataset includes 15,000 customer records with demographic, credit, and financial attributes such as income, credit score, loan amount, debt-to-income ratio, and asset value.

The project involves data preprocessing, handling missing values, exploratory data analysis (EDA), and visualization to understand key risk patterns. It helps identify factors influencing financial risk and supports better decision-making for credit approval and risk management.

Key Techniques Used:

Data Cleaning and Imputation
Exploratory Data Analysis (EDA)
Statistical Summary and Visualization
Correlation Analysis
Risk Category Analysis

Tools & Libraries:
Python, Pandas, NumPy, Matplotlib, Seaborn
